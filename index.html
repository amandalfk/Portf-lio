Cursos e Certificados

Curso de Desenvolvimento de Sistemas - SENAI Manuel Garcia Filho

Estou cursando o Curso Técnico em Desenvolvimento de Sistemas no SENAI Manuel Garcia Filho que tem por objetivo habilitar profissionais para desenvolver, testar e implantar sistemas computacionais, atendendo normas e padrão de qualidade, usabilidade, integridade e segurança da informação. O curso abrange os seguintes tópicos:

Programação e Algoritmos

Lógica de programação, algoritmos, ambientes de desenvolvimento, Git e versionamento em nuvem.

Sistemas Operacionais

Tipos de SO (fechado e aberto), redes, firewall, VPN e segurança cibernética.

Levantamento de Requisitos

Definição, documentação e gerenciamento de requisitos com Scrum, Kanban e Design Thinking.

Arquitetura de Redes com IoT

Estrutura de redes, protocolos de comunicação e dispositivos IoT.

Banco de Dados

Sistemas de gerenciamento de banco de dados (SGBD) e modelo relacional.

Linguagem de Marcação

HTML, CSS e criação de páginas web.

Programação Back-End

JavaScript, design responsivo, UX/UI e frameworks.

Programação Front-End

Sistemas de gerenciamento de banco de dados (SGBD) e modelo relacional.

Programação para Dispositivos Móveis

Desenvolvimento de apps móveis, APIs e integração com RESTful services.

Internet das Coisas (IoT)

Automação, protocolos de comunicação e interfaces interativas.

Testes de Software

Planejamento e execução de testes para garantir qualidade.

Projetos de Software

Metodologias ágeis, gerenciamento de projetos e padrões de qualidade.

Conclusão: Dezembro de 2025
